# React Jotai Atom Product Listing

This is a minimal React application using Jotai for state management and Tailwind CSS for styling. It shows a simple product listing with search.

Getting started:

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

3. Open the URL shown by Vite (usually http://localhost:5173)
